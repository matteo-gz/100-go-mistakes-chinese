# 数据类型

## 八进制混淆
即代码相对人类可读性而言
```go
fmt.Println(100 + 010)  //结果 108
fmt.Println(100 + 0o10) //结果 108
fmt.Println(100+010 == 100+0o10) // true
```
八进制表达最好用`0o`前缀

二进制用`0b`前缀
十六进制用`0x`前缀

## 整数溢出
当一个数据超过范围,每次需要增加额外逻辑来判断处理是否溢出。
我个人觉得是一种损耗,应该是你数据类型范围定义错误了。
